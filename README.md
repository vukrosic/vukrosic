#### 🤖 AI Models

Tiny, honest, code-trained models — each does one narrow thing a script can't, runs on a CPU in milliseconds, and ships with its own benchmark, tests, and technical report.

- **[nano-proofread](https://github.com/vukrosic/nano-proofread)** — fixes context errors a spell-checker can't see (`their` → `they're`). → [HF](https://huggingface.co/vukrosic/nano-proofread)
- **[nano-g2p](https://github.com/vukrosic/nano-g2p)** — predicts how a word is pronounced (`though` → `DH OW`). → [HF](https://huggingface.co/vukrosic/nano-g2p)
- **[nano-spell](https://github.com/vukrosic/nano-spell)** — corrects a misspelled word to the one it meant (`recieve` → `receive`). → [HF](https://huggingface.co/vukrosic/nano-spell)
- **[nano-vowel](https://github.com/vukrosic/nano-vowel)** — restores the missing vowels to a word (`wrld` → `world`). → [HF](https://huggingface.co/vukrosic/nano-vowel)
- **[nano-case](https://github.com/vukrosic/nano-case)** — splits separator-less identifiers into any case style (`sdkmodel` → `sdk_model`). → [HF](https://huggingface.co/vukrosic/nano-case)
- **[nano-dates](https://github.com/vukrosic/nano-dates)** — turns natural date phrases into ISO-8601 dates (`next friday` → `2026-07-03`). → [HF](https://huggingface.co/vukrosic/nano-dates)
